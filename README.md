# Sideline

The AI marketing employee for sports coaches. A coach uploads raw training
footage; Sideline finds the best moments, cuts vertical clips with burned-in
captions in the coach's brand color, and writes the caption, hook, hashtags,
and CTA for each.

- **SPEC.md** — the V1 build contract
- **PROGRESS.md** — what's done, what's next, how to test
- **IDEAS.md** — parking lot for out-of-scope ideas

## Quick start

```bash
npm install
cp .env.local.example .env.local   # Demo Mode is on by default
npm run dev                        # open http://localhost:3000
```
